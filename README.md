# Pet Food Shop Website

A simple, responsive website for a pet food shop built using HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Adapts to different screen sizes using CSS media queries.
- **Product Listings**: Displays 5 pet food products with images, detailed descriptions, and prices.
- **Interactive Elements**: Add to cart buttons with JavaScript alerts for user feedback.
- **Navigation**: Smooth scrolling navigation to different sections of the page.
- **Clean Layout**: Header with navigation, main content sections, and footer.

## Technologies Used

- **HTML**: Structure and content
- **CSS**: Styling and responsive layout
- **JavaScript**: Interactivity for cart functionality

## Files

- `index.html`: Main HTML file containing the structure and content
- `style.css`: CSS file for styling the website
- `script.js`: JavaScript file for handling user interactions

## How to Run

1. Clone or download the project files.
2. Open `index.html` in a web browser.
3. Navigate through the sections and interact with the products.
